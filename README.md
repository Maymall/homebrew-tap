# Maymall's Homebrew Tap

Homebrew formulae for [Maymall](https://github.com/Maymall)'s tools.

## Install

```bash
brew install Maymall/tap/rgfile
```

Or tap first:

```bash
brew tap Maymall/tap
brew install rgfile
```

## Formulae

| Formula | Description |
|---|---|
| [`rgfile`](Formula/rgfile.rb) | Fast, robust command-line client for [GigaFile.nu](https://gigafile.nu) — [source](https://github.com/Maymall/gigafile-rust-cli) |

Formulae install prebuilt release binaries (macOS arm64/Intel, Linux x86_64
static musl) verified against the release SHA-256 checksums.
